# Practical 3: Excel EDA with Formulas, Pivot Tables and Charts

## Objective

The objective of this practical was to perform Exploratory Data Analysis (EDA) using Excel by applying formulas, pivot tables, slicers, and charts to derive meaningful insights from the dataset.

---

## Tasks Performed

1. Created the file **p3_excel_eda_dashboard.xlsx** in the 02_excel folder.

2. Calculated summary statistics for key numeric columns such as Sales, Profit, and Quantity:

   * Count
   * Sum
   * Average
   * Minimum
   * Maximum
   * Median
   * Standard Deviation

3. Created Pivot Tables:

   * **Pivot Table 1:** Performance analysis by Category and Segment
   * **Pivot Table 2:** Performance analysis by Region, State, and City
   * **Pivot Table 3:** Sales trend by Month/Year

4. Added visualizations:

   * Bar chart for category-wise performance
   * Pie chart for segment distribution
   * Line chart for monthly/yearly trends

5. Inserted slicers:

   * Category slicer
   * Region slicer

6. Designed a Dashboard sheet:

   * Added title: *Sales Performance Dashboard*
   * Created KPI cards (Total Sales, Total Profit, Order Count)
   * Added charts with proper labels and formatting

---

## Observations & Insights

* Technology category showed the highest sales performance compared to Furniture and Office Supplies.
* The Consumer segment contributed the largest share of overall sales.
* Certain regions (like West/East) performed better in terms of profit compared to others.
* Sales showed an increasing trend over time with some seasonal fluctuations.

---

## Report Questions

### 1. Which pivot table gave the most useful insight?

The pivot table showing **performance by Category and Segment** provided the most useful insight because it clearly highlighted which product categories and customer segments contributed the most to sales and profit. This helped in understanding business performance at a high level.

---

### 2. What changed when you used slicers?

When slicers were applied, the dashboard became interactive. It allowed filtering data dynamically based on category and region, making it easier to analyze specific segments without modifying the original data. This improved data exploration and decision-making.

---

### 3. What is one limitation of doing this only in Excel?

One limitation of using Excel for EDA is that it becomes difficult to handle very large datasets and advanced analytics. Excel dashboards are also less scalable and less interactive compared to tools like Power BI or Python-based analysis.

---

## Conclusion

This practical helped in understanding how Excel can be used for basic data analysis and dashboard creation. Pivot tables, charts, and slicers made it possible to transform raw data into meaningful insights effectively.
