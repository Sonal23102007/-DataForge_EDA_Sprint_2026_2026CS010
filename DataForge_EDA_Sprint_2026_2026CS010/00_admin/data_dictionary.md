# Data Dictionary

## Dataset: Superstore Sales

| Column Name   | Data Type | Description                      |
| ------------- | --------- | -------------------------------- |
| Row ID        | Integer   | Unique ID for each row           |
| Order ID      | String    | Unique identifier for each order |
| Order Date    | Date      | Date when order was placed       |
| Ship Date     | Date      | Date when order was shipped      |
| Ship Mode     | String    | Shipping method                  |
| Customer ID   | String    | Unique customer ID               |
| Customer Name | String    | Name of the customer             |
| Segment       | String    | Customer segment                 |
| Country       | String    | Country name                     |
| City          | String    | City name                        |
| State         | String    | State name                       |
| Postal Code   | Integer   | ZIP/Postal code                  |
| Region        | String    | Region name                      |
| Product ID    | String    | Product identifier               |
| Category      | String    | Product category                 |
| Sub-Category  | String    | Product sub-category             |
| Product Name  | String    | Name of the product              |
| Sales         | Float     | Sales amount                     |
