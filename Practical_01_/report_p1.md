## Report question-Practical 1
## 1. What is one row in this dataset representing?
Ans: Each row in the dataset represents a single product-level sales transaction.
It contains details about the order, customer, product, location, and the sales amount for that specific item.

## 2. Which columns look numerical, categorical and date/time?
Ans:Numerical: Row ID, Postal Code, Sales
Categorical: Customer Name, City, Category, Product Name, etc.
Date/Time: Order Date, Ship Date

## 3. What can go wrong if a team starts analysis without a data dictionary?

Answer:

If a team starts analysis without a data dictionary, several issues can occur:

 Misunderstanding of column meanings
 Wrong data type assumptions (e.g., treating dates as text)
 Incorrect analysis and misleading results
 Confusion between similar columns (e.g., Order ID vs Product ID)
 Poor communication among team members
Data cleaning errors
